# Town Of Us Mira: Unity

This repository contains the Unity project for [Town Of Us Mira](https://github.com/AU-Avengers/TOU-Mira).

Some assets are not included in this repository, as they are copyrighted and cannot be distributed. Please contact us if you need access to them.

In order to use this project, make sure to use Unity 2022.3.44f1.
You will also need to get the Android Build Support module through Unity Hub to compile the bundle for android (which will be functional very soon!)
